# VIPER

A simple framework of VIPER components to construct modules.

## Swift Package Manager

Add to your `Package.swift`'s `dependencies:` array:

```swift
.package(url: "git@github.com:thomverbeek/VIPER.git", from: "0.3.0"),
```
